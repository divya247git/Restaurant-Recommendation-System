# 🍽️ Restaurant Recommendation System

A restaurant recommendation application that helps users discover restaurants based on their preferred cuisine, location, and ratings. The system uses a hybrid recommendation approach to provide relevant and personalized suggestions.

---

## 📌 Features

- Recommend restaurants based on cuisine preferences
- Location-based restaurant search
- Rating-based restaurant ranking
- Personalized recommendations using a hybrid recommendation system
- Real-time restaurant data integration through the Zomato API

---

## 🛠️ Tech Stack

- Python
- Zomato API
- Hybrid Recommendation System
- REST APIs
- JSON

---

## ⚙️ How It Works

1. User enters preferences such as cuisine and location.
2. The system fetches restaurant data using the Zomato API.
3. Restaurants are filtered and ranked based on ratings and relevance.
4. Personalized recommendations are displayed to the user.
